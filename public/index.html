The Chat Client

Download the chat client as an executable jar file (requires Java). To test the client by yourself, start more than one instance (each with a unique user name)

This is done by left double-clicking on the downloaded .jar file.

To hook up enter URL, port and chosen UserID and click 'Connect'

To chat enter UserID of receiver and message to send and click 'Send'
